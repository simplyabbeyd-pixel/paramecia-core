# Presentation Sources

PowerPoint is the primary presentation artifact for the Paramecia / Triquel Living Atlas.

This directory stores versioned presentation inputs rather than exported binaries by default:

- slide outlines
- diagram references
- source citations
- canon-status notes
- asset manifests
- release notes

## Tool Roles

- **Beautiful.ai** supplies structured diagram drafts and alternate layouts.
- **Mermaid** supplies versioned relationship and workflow diagrams.
- **Figma / FigJam** supplies selective visual experiments.
- **Ace Knowledge Graph** may supply dense relationship maps after canon stabilization.
- **PowerPoint** integrates the final narrative, diagrams, artwork, and accessibility treatment.

Generated presentation content remains provisional until checked against canon files and the Three Accord gates.

## Required Release Notes

Each major deck should record:

1. title and version
2. canon files used
3. diagrams used and their source paths
4. known provisional material
5. unresolved contradictions
6. final output link
7. release date
