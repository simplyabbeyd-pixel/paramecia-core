# World Stack & Lore Index

This directory holds the living pieces of the continuity, named so we can grab them fast.

## The Worlds
* **Paramecia:** Homebrew universe and parent continuity.
* **Triquel:** The active build surface: factions, rites, maps, systems, conflicts, and what the world wants next.
* **Hells Branch:** Consent, power, responsibility, rescue logic, and the dangerous habit of becoming infrastructure.
* **Whisperwood:** Soft place, sharp implications. Track what makes it cute, why it persists, and what it protects.
* **Gemini:** Visual imagination, moodboards, character silhouettes, scene seeds, and alternate interpretations.
