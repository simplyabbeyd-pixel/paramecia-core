# Timepoint 0 Systems

This folder holds the models that explain how ecological origin becomes movement, settlement, exchange, and civilization.

## Movement taxonomy

### Stayers
Rooted peoples who protect and stabilize a place.

Examples: Dervkin, Bearkin, Hedgekin.

### Seasonal movers
Peoples whose return cycles carry relationships between biomes.

Examples: Rabbitfolk, Pollinatorkin, some Túlkin.

### Edge-walkers
Peoples who adapt, translate, and bridge between systems.

Examples: Foxkin, Catkin, Ratfolk.

### Transition specialists
Peoples who witness, cross, or transform thresholds.

Examples: Ravenkin, Mothkin, certain Faust-type figures.

### Biome-process peoples
Process-first peoples whose identity is inseparable from landscape activity.

Examples: Swampfolk / Sloamfolk.

## First settlement logic

A first settlement must have more than population density. It begins where several pressures overlap:

1. **Safety** — protection from harm
2. **Trade** — exchange creates a reason to meet
3. **Shared resources** — water, soil, food, shelter
4. **Threshold crossing** — paths and biomes intersect
5. **heaRTH** — warmth, return, care, and story make a place worth coming back to

## Development order

1. Lock who moves and why.
2. Map seasonal and edge movement routes.
3. Identify ecological exchanges created by those routes.
4. Find recurring threshold crossings.
5. Test which crossings can support safe return.
6. Place the first settlement only after those conditions exist.

## Required questions

- What makes somewhere worth staying?
- What returns cyclically?
- What happens between places?
- What happens when something changes?
- How does landscape itself move?
- Why does the first village exist here and nowhere else?
