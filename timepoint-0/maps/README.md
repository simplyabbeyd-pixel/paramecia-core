# Timepoint 0 Maps

This folder tracks the approved visual atlas sequence and future source assets.

## Current map set

1. **The First Noticed World** — origin biomes and starting peoples
2. **Kin & Animal Relationship to Biome Types** — home biomes, secondary biomes, ecological gifts, edge specialties
3. **Friendships & Alliances** — early local bonds and exchange routes
4. **Deep Friendships & Kin Relations** — durable chosen relationships
5. **First Cross-Species Kin** — named crosskin traditions arising at biome edges
6. **Triquel Origin Stack** — relationship between Timepoint 0 and later movement, settlement, domain, character, lantern, and tarot systems

## Asset rules

- Preserve the original uploaded image outside Git history until a stable asset naming and licensing workflow is chosen.
- Store text extracted from images in Markdown or JSON beside future image assets.
- Do not treat generated typography as authoritative without checking it against canon text.
- Map lines must indicate a defined relationship type, ecological exchange, movement path, or threshold function.
- Every map should state its timepoint and canon status.
