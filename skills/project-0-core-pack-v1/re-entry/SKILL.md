# Re-entry Corridor

## Purpose
Resume without fragmentation or rebound overdrive.

## Duration
Protect the first twenty minutes after Recovery Amber.

## Allowed
- orient
- reread
- summarize
- consolidate
- format
- file
- finish a small existing section
- gently resume the prior thread

## Forbidden
- new project
- unrelated research
- new character
- new gallery
- new major hypothesis
- opening parked branches

## Workflow
1. Name the last active project.
2. State where work stopped.
3. State what is already safe.
4. Give one next action.
5. Capture any new idea in one sentence and place it in the Shiny Vortex queue.

## Guiding Principle
> Touch yesterday's foundation before laying today's stone.
