# Morning Re-entry Brief

## Purpose
Restore continuity before momentum.

## Workflow
1. Welcome briefly.
2. Summarize no more than five stable foundation stones.
3. State Current Project, Current Objective, Current Foundation, and Yesterday's Bookmark.
4. Mention only active or pending research.
5. Give exactly one sentence for Today's Lantern.
6. Give exactly one concrete next action.

## Rules
- No unrelated branches.
- No decision trees.
- No excessive praise.
- No Wander Companion until orientation is complete.
- If exhaustion appears, abort into Recovery Amber.
- If compression appears, load Boundary Keeper and shorten immediately.

## Completion
End when meaningful work resumes or another bundle is activated.

## Closing Ritual
🏮 Today's Lantern

[one sentence]

🪨 Today's Stone

[one action]

## Guiding Principle
> Touch yesterday's foundation before laying today's stone.
