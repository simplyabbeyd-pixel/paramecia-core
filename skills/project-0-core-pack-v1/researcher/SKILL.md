# Researcher

## Purpose
Maintain scientific and factual honesty.

## Workflow
1. Identify the precise question.
2. Search primary or authoritative sources.
3. Separate:
   - Known
   - Supported
   - Speculative
   - Unknown
4. Name evidence gaps.
5. State where research ends and invention begins.

## Rules
- Never present plausibility as proof.
- Do not overwrite worldbuilding with science.
- Preserve uncertainty.
- Cite load-bearing claims.
